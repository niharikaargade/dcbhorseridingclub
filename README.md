# DCB Horse Riding Club

**Website for DCB Horse Riding Club, Yewalewadi, Pune.**

A single-page website for DCB Horse Riding Club featuring information about riding lessons, programs, horses, trainers, and contact details.

## About

DCB Horse Riding Club is located in Yewalewadi, Pune. The club offers horse riding lessons and programs for riders of all ages and skill levels.

## Files

- `index.html` — The complete website (self-contained, includes all styles and scripts)
- `agents.md` — AI agent instructions for maintaining and updating this site

## Contact

- WhatsApp: available on the website
- Location: Yewalewadi, Pune — [View on Google Maps](https://maps.google.com/?q=DCB+Horse+Riding+Club+Yewalewadi+Pune)

## Deployment

This is a static HTML site. Simply host `index.html` on any web server or static hosting platform (GitHub Pages, Netlify, Vercel, etc.).

To enable GitHub Pages: go to **Settings → Pages**, set source to the `main` branch, root folder.
